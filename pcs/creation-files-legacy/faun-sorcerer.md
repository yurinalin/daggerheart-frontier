# Oráculo das Besteiras - Murillo

### Esboço de Personagem
Oráculo das Besteiras — Faun (Fauno) Feiticeiro, Wild Channeler
Um fauno com chifres tortos que "canaliza" magia selvagem, mas nunca sabe o que vai acontecer. Ele mesmo tem medo dos próprios feitiços. Prediz o futuro, mas todas as previsões são coisas inúteis tipo "amanhã vai ser quinta-feira" ou "alguém nesta sala vai espirrar". Surpreendentemente, ele está sempre certo. Comunidade: Wildborne — porque nenhuma civilização organizada o aceita como oráculo oficial.
