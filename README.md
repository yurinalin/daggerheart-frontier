# Daggerheart Frontier Campaign

## Estrutura
- `lore/`: verdade do cenário
- `sessions/`: preparação e resumos
- `npcs/`: fichas rápidas de NPCs
- `locations/`: locais importantes
- `sandbox/`: tabelas e improviso
- `foundry-publish/`: arquivos prontos para enviar ao Foundry

## Fluxo
1. Escrever ou atualizar material nas pastas principais
2. Copiar/publicar o que for útil para `foundry-publish/`
3. Sincronizar com o Foundry
4. Após sessão, atualizar `sessions/` e commitar

## Regras pessoais
- Foundry não é a fonte de verdade
- Markdown primeiro, Foundry depois
- Toda sessão gera recap
